# Getting Started

## Installation

Inside your React project directory, run the following:

```bash
yarn add react-flex-ready
```

Or with npm:

```bash
npm install react-flex-ready
```

## Features

- TypeScript support
- Flexbox ready
- Easy to use
- Easy to customize

```jsx
import myList from "some-list";

const YourApp = () => (
  <Flex
    col={4}
    colTablet={6}
    colMobile={12}
    gap={1}
  >
    {myList.map(({ columns }, i) => (
      <Item
        key={i}
        col={columns}
        colTablet={6}
        colMobile={12}
        gap={1}
        marginBottom={30}
        stretch
      >
        <div
          style={{
            textAlign: "center",
            width: "100%",
            border: "1px solid #eee",
            background: "#eee"
          }}
        >
          <p>{columns}</p>
        </div>
      </Item>
    ))}
  </Flex>
);
```
